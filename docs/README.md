# noti. 문서

이 폴더는 `noti.`의 제품 의도와 구현 결정을 기록한다. 코드와 함께 갱신하며, 아직 구현되지 않은 내용은 계획으로 명시한다.

## 문서 안내

- [제품 정의](product.md): 해결하려는 문제, 핵심 경험, MVP 범위
- [아키텍처](architecture.md): 알림 수집부터 화면 표시까지의 기술 설계
- [중요도 정책](importance-policy.md): 중요 알림을 판정하고 학습하는 규칙
- [개발 계획](development-plan.md): 구현 순서와 단계별 완료 조건

## 현재 상태

- Jetpack Compose 기본 프로젝트 생성 완료
- 패키지명: `com.hanseo.noti`
- 본 기능은 아직 구현 전
- 체크인/일일 브리핑과 온디바이스 AI는 MVP 이후 검토

## 문서 원칙

- 실제 구현과 계획을 구분한다.
- 정책 수치와 구조가 코드에서 달라지면 문서도 함께 수정한다.
- 실제 사용자의 알림 본문, 이름, 연락처 등 개인정보는 예시나 테스트 데이터로 커밋하지 않는다.
